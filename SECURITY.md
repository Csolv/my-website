# Security Policy — Asociación Deportiva Curundú

## Reporting a vulnerability
If you discover a security issue with the ADC website (https://adcurundu.org),
please report it privately by email to **adcurundu@gmail.com** with the subject
line "Security". Please include steps to reproduce and do not publicly disclose
the issue until we have had a reasonable chance to address it.

We are a small volunteer-run nonprofit and do not operate a paid bug-bounty
program, but we are grateful for responsible disclosure and will credit
reporters who wish to be acknowledged.

## Scope
- In scope: the public website (adcurundu.org) and this repository.
- Out of scope: third-party services we embed (GoFundMe, Google Forms, Google
  Calendar, Behold/Instagram, Google Translate). Report issues with those to
  the respective provider.

## Supported
This is a static site; the live version is whatever is deployed from the
default branch.
